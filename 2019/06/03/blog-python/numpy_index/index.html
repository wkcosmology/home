Numpy中的index方法入门

Posted on 2019-06-03 | In Blog , ComputerScience
这里讲解了一些Numpy中index的基本操作。

整数索引

对于一个n维的数组，如果某一个维度的索引是整数，那么这个维度就会消失，也就是产生的数组维度减一；
如果所有的维度都是整数，那么返回的就是这个数组dtype类型的一个对象。

整数索引返回的是引用。
如果所有的维度都是整数，并且数组的元素是不可变的，那么返回的不能看做引用；如果数组的元素是可变的，那么返回的是引用。
