我的vim配置

Posted on 2019-06-03 | In Blog , ComputerScience
这篇文章主要介绍我是如何配置我的vim的，也是警戒自己以后不要再随便折腾vim了，现在已经很好了。

我的vim配置主要是基于Spacevim，加上一些自己的配置。我先介绍一下除了Spacevim默认插件之外的几个我经常使用的插件。

插件介绍

自动补全插件：YouCompleteMe

其中目前vim上的补全插件非常多，比如deoplete，coc等。我使用YCM的原因是我主要用的语言只有C&C++和Python，而YCM对这二者的支持非常好，速度又很快，所以我优先使用YCM。我个人感觉deoplete和coc的对上面三种语言的补全速度没有YCM快，不过我没有经过严格测试，各位有兴趣可以测试一下。YCM饱受诟病的一点就是安装麻烦，这一点我是承认的，所以我下面大概总结一下安装过程，不过我的经验可能不够全面，如果遇到其他问题，还是需要google一下的。

  1. 首先我们在spacevim中enable了YCM，即enable_ycm=1（见下方配置），再次进入vim的时候，就会自动下载vim插件，（我的spacevim自动下载的插件存储在~/.cache/vimfiles/repos/github.com/目录中）；

  2. 由于YCM还依赖于一些第三方库，所以我们需要进入该目录，即~/.cache/vimfiles/repos/github.com/Valloric/YouCompleteMe，运行git submodule update --init --recursive。注意，这时候可能会撞到墙，因为其中一个go语言的第三方包被墙了，如果你遇到这种问题，网上的一种方法是(我没有尝试过，不过这是我能找到的唯一的办法了)：

    • 在YCM的文件夹中执行git submodule update --init（注意没有recursive）；

函数签名显示插件-Shougo/echodoc.vim

对于很多第三方库的Python函数来说，由于可能参数会有很多而我们很难记住，这时候最好能够在我们输入参数的时候显示函数的形参列表，而Shougo的这个插件就是做这件事的。至于函数签名的位置，也可以设置成为悬浮窗口，因为这个可能会和自动补全冲突，所以我选择显示在下方。

模糊搜索插件-FZF

模糊搜索插件目前有很多，比如CtrlP、Unite、FZF、Denite等，其中FZF是最容易上手、学习成本最低的。它具有以下功能：

  1. FzfColors: 切换主题
  2. FzfFiles: 搜索当前目录下面的文件
  3. FzfMru: 搜索vim最近访问过的文档
  4. FzfOutline 搜索当前buffer中的函数、类、全局变量
  5. Fzfbuffers: 搜索当前vim打开的buffer
  6. FzfRegister: 搜索寄存器
  7. FzfTags: 搜索tags，对于C&C++而言，是ctags建立的项目tags；对于Python而言，就是当前库中的tags，即所有的变量、包等
  8. FzfJumps: 搜索跳转表
  9. FzfQuickfix: 搜索quickfix表

框架管理插件-tagbar

利用这个插件，我们可以看到当前buffer中的函数、类、类方法、全局变量等。利用这个，我们可以很方便地跳转到不同的位置，也可以清楚地看到当前文件中的各种元素。我们前面提到的Fzfouline也能实现类似功能，但是tagbar的好处是它一直在这，能让我看清楚整个文件的结构。

括号快捷操作插件-vim-surround

这个插件主要是为了增加、删除和修改括号的，其中括号包括(), [], {}, “”, ‘’, <>等。这个插件的强大之处在于其和文本对象的配合，以及和可视化模式和f、F键的配合。我们需要注意一点：左括号在增加和修改的时候，会在括号中增加一对空格；有括号则没有多余的空格产生。
对于删除和修改括号功能，它直接对于当前光标位置外围第一个括号进行操作，所以不需要选中文本，都是很简单的操作，比如

  1. cs"'：把"Hello world!"改成'Hello world!'
  2. cs'<q>：把'Hello world!'改成<q>Hello world!</q>
  3. cst"：把<q>Hello world!</q>改成"Hello world!"
  4. ds"：把"Hello world!"改成Hello world!
    对于增加括号，我们需要选中文本，在进行操作，而选中文本有两种方式，第一种利用文本对象，比如
  5. ysiw]：把Hello world!改成[Hello] world!，注意：光标在Hello上
  6. ysiw{：把Hello world!改成{ Hello } world!
  7. yss)：把Hello world!改成(Hello world!)(注意，这个是对整行进行操作)
    第二种是利用可视模式（此时配合f、F键会更方便），比如
    Hello world, long time no see，此时光标在H上，我们需要在Hello world周围加上括号，操作步骤：
    • v进入可视化模式；
    • fd搜索到右边第一个字母d，这时候可视化模式也会选中相应区域；
    • S这个键位是vim-surround中对可视化选中区域进行操作时使用的键；
    • )加上括号。
      除了对括号进行操作之外，vim-surround还支持增加函数，也就是增加圆括号的同时在圆括号之前加上函数名，比如：
      1. ysa"fstr：把"Hello world!"改成str("Hello world!")

快速跳转插件-vim-easymotion

我们目前可以使用的跳转方式只有几种：上下左右键、fF键和搜索功能。但是这些都不是很方便，fF键需要从当前位置开始跳转，如果常用的字母，可能需要多次跳转才能到达；搜索功能也是类似的问题，它只能顺序跳转。easymotion支持我们利用一个或者两个字母，在当前屏幕中快速跳转。它会在所有目标字符处利用很容易输入的字符组合来进行定位，你只需要输入相应字母，就可以快速跳转了。

TOML是一种旨在成为一个小规模、易于使用的语义化的配置文件格式，它被设计为可以无二义性的转换为一个哈希表。
“TOML”这个名字是“Tom’s Obvious, Minimal Language（汤姆的浅显的、极简的语言）”的首字母略写词[2]。“Tom”指它的作者Tom Preston-Werner。
TOML的语法广泛地由key = “value”、[节名]与#注释构成。
它支持以下数据类型：字符串、整形、浮点型、布尔型、日期时间、数组和图表。
