instance method, static method and classmethod 的区别

Posted on 2018-09-14 | In Note , Python
主要介绍instance method, static method and classmethod 的区别
  • instance method: 是对象的方法，只能通过对象调用，申明的时候不需要装饰器；
  • class method: 是类方法，可以通过对象或者类调用，申明的时候需要装饰器，class method 申明的时候第一个参数总是该类，在调用class method的时候不需要输入这第一个参数；
  • static method: 和普通函数类似，只是需要通过类名称来调用，这个可以避免全局命名空间污染。
