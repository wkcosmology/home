计算机书籍推荐

Posted on 2018-09-14 | In RecommendList
这篇文章主要记录一些自己读过的或者想读的计算机方面的书籍，并且读过的书籍会给出一些书评，这个会不断更新，也督促自己多读书。

YOU MUST LEARN

Coding style

  • [x] The practice of programming
    这本书很神奇。每一章用最简短的篇幅讲解清楚了一个专题。第一章讲了每个人应该遵守的最基本的编程风格；第二章非常简洁而清晰地讲解了几个常用的数据结构和算法；第三章用一个马尔科夫链的例子来介绍了如果去设计一个程序，将一个程序子程序化，以及不同的语言的风格导致程序在编写难度和运行速度上的差异；第三章专门去讲解了程序的封装和接口的设计；第四章讲解了作者debug的一些经验；第六章讲解了如何去测试你的程序，并且告诉你测试程序对程序设计的重要作用；第七章介绍了一些性能调优的经验，也讲了性能调优的必要性和不必要性，尤其是不必要性，这个被多次提起过，就是不要一味追求性能调优，而是在需要的时候再去考虑性能问题；第八章和第九章没有细读，暂时也不能领会其中妙处。
  • [ ] The code complete
  • [ ] Refactoring: improving the design of the existing code

Algorithm

  • [x] Algorithms
    这本书用java为使用语言，以算法中的排序、查找、图和字符串为顺序来写的。在讲解算法的同时也对数据结构中的链表、树、图做了很好的介绍。这本书在coursera上有配套的课程，也有配套的网站，有丰富的辅助资源。
  • [ ] Algorithms(DPV)
  • [ ] Introduction to algorithms

Python

  • [x] Fluent python
    这本书是python的进阶书籍，对Python的特性进行了很细致而有趣的介绍。讲解很清晰，示例代码很多而且很明白。这本书主要是讲解python本身的讲解，也涉及一些内部库的介绍。
