撤销修改，撤销暂存区添加，版本库回退

git一个重要的功能就是能够回退到以前的状态来实现备份功能。对于git而言，我们有三个层次的回退：

  • 撤销修改：使用暂存区的文件刷新文件夹中的文件
  • 撤销暂存区添加：使用版本库里面的文件来刷新暂存区文件
  • 版本库回退：使用以前的版本来替代目前的版本
