《恒星物理》第七章：恒星演化及致密天体I--小质量恒星的演化

Posted on 2018-12-23 | In Note , StellarAstrophysics
本文主要介绍了小质量恒星从形成之后的演化过程。

主序前演化

Hayashi线

我们在第二章中介绍过恒星时从星际气体云坍缩过来的，形成了原恒星。在恒星赫罗图中大约有效温度3000-5000K范围内，有一条与主序几乎垂直的线，称为Hayashi线。 Hayashi线有两个特征：

  1. 由星际云凝聚坍缩形成的原恒星，当它的内部建立起流体静力学平衡，并且完全对流，可以称为主序前恒星时，它在HR图上将落在Hayashi线上；
  2. 在给定恒星的质量和化学组成时，HR图将以Hayashi线为界，按照天体是否满足流体静力学平衡条件，划分为“禁止“和”允许“两个区域，Hayashi右边的区域，不满足流体静力学平衡条件，称为”禁止“区域。

定性描述恒星从Hayashi线到主序的演化：

  1. 由引力能补充辐射掉的热能并提高中心温度；
  2. 演化的时标是：$t _ { K H } = \frac { G M ^ { 2 } } { R L }$；
  3. 沿Hayashi线住下走，半径减小，光度减小；
  4. 内部出现辐射平衡区后，Hayashi线向左移动，中心温度持续增高；

主序

恒星第一次达到主序并开始平衡的氢燃烧阶段叫做零龄主序(ZAMS)。达到零龄主序的时间与恒星的质量成反比。通过观测，我们发现绝大部分恒星在HR图中的位置落在一条窄带上，这条窄带被称为主序带。主序星一般有如下性质：

  1. 均匀的化学组成；
  2. 核心氢燃烧，完全流体静力学平衡；
  3. 质量范围：$0.08 M _ { \odot } < M < 100 M _ { \odot }$；
  4. 质光关系：$L \sim M ^ { 2.5 - 4 }$；
  5. 质量-半径关系：$R \sim M ^ { 0.5 - 1 }$。

主序星的内部结构

主序星内部一般有三个区域：核反应燃烧核, 对流区以及辐射区。对于大质量恒星，能量主要来自对温度很敏感的CNO循环。因为能量产生率随半径快速变化，辐射不能有效地将能量传递出去，所以会导致对流的产生，而在外部，能量传输变得有效起来，就是辐射区。对于小质量恒星，PP链是能量的主要来源，这时候内部就是辐射区，而在表面，由于氢电离区的存在，会导致不透明度增加，所以表面会是对流区。对于更小质量的恒星而言，对流区的范围逐渐扩大，直到核反应燃烧核之外就全是对流区。

主序阶段的演化时标就是核反应时标

因为质光关系的幂指数大于1，所以恒星质量越大，这个时标越短，也就越快离开主序。主序阶段终止的标志有两种：

  1. 核心产生一个等温氦核；
  2. 氢燃烧继续在环绕着氦核的壳层进行。

亚巨星阶段

亚巨星阶段也就是恒星从主序到巨星过度的一个中间状态，其主要特征：

  1. 核心氢烧完、枯竭，形成氦中心核和氢丰富的外层；
  2. 中心温度降低，氦尚没点燃，而外壳氢燃烧，中心氦核增大；
  3. 氦核收缩，壳层氢燃烧，体积膨胀，表面温度降低。
  4. 在H-R图表现为恒星逐渐向右脱离主序。

红巨星支

红巨星阶段的主要特征：

  1. He核进一步收缩,中心温度上升，核区电子简并；
  2. 半径增大，壳层H燃烧，光度增加，膨胀使得温度下降；
  3. 恒星包层产生对流（类似Hayashi线）对流区不断向恒星内部扩展；
  4. 对流不断将内部的He搬运到外层，He核质量不断增大；
  5. 在HR图上向右上方攀升成为红巨星；
  6. 在红巨星分支的顶端，核心的电子完全简并，压强P主要是简并的电子压强, 它与温度T无关；

我们可以看到

  1. 小质量恒星的光度L仅仅与氦核的质量以及半径有关，而与恒星的总质量无关，因此它们不遵守主序的质光关系；
  2. 当壳层源燃烧使中心核$M_c$增大时，$R_c$减小，$T_c$增大，造成L增大又使$M_c$增加得更快；
  3. 中心核收缩释放引力能，加热了壳源及中心核。

当He核达到$10^8K$的时候就会点燃。根据温度和He核质量和半径的关系，通过理论计算可以得到，当$M_c > 0.45M_{\odot}$的时候就会发生He的核反应。

质量$< 0.5 M_{\odot}$的恒 星永远不会达到$M = 0.45 M_{\odot}$,故不会发生He燃烧，在丢失 掉全部外壳后变成He白矮星。

He的核反应是3$\alpha$反应，发生在非常致密的核心区，是一个非常快速和剧烈的反应，称为氦闪。在氦闪的过程中，核心温度在几秒内上升，使得电子不再保持简并，而是遵从Maxwell分布律。

水平分支

水平分支具有以下特点：

  1. He $\rightarrow$ C $\rightarrow$ O 在核心发生聚变；
  2. 壳层的氢发生燃烧；
  3. 在HR图上表现为恒星向左下方移动至水平支；
  4. 星核膨胀、吸热，光度剧减

渐进巨星支(AGB)

渐进巨星分支的特点是：

  1. 中心氦耗尽，变成C、O核，核反应停止，开始收缩；
  2. 收缩升高外层压力和温度，核心外层又有一层氦点火；
  3. 氦燃烧壳层外的氢壳层在继续燃烧；
  4. 双壳层产能使得恒星光度大增，外包层膨胀；
  5. 在HR图上表现为恒星向右上方再次攀升成为红巨星。

行星状星云

行星状星云是低质量恒星在死亡时抛出的气体包层，受到中心高温的白矮星发射的紫外光子电离辐射而发光而产生的。
