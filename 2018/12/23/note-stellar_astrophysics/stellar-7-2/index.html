《恒星物理》第七章：恒星演化及致密天体II--中大质量恒星演化

Posted on 2018-12-23 | In Note , StellarAstrophysics
本文主要介绍了中大质量恒星的演化过程。

中等质量恒星的演化

中等质量恒星（$2M _ { \odot } < M < 8M_ { \odot }$）的演化与小质量恒星不同的是：

  1. 恒星内部的H燃烧通过CNO循环进行，内部温度更高；
  2. 主序寿命更短，演化到红巨星分支时间也非常短；
  3. He核不再是简并的，C和更重元素的燃烧可平稳进行；
  4. 核心区核反应产生的能量主要以对流的方式向外传递；
  5. 外层中氢和氦电离区存在脉动，多次穿过造父脉动带；
  6. 演化的最终产物可能是碳氧白矮星或氧氖镁白矮星；

大质量恒星的演化

中等质量恒星（$M > 8M_ { \odot }$）的演化的特点是：

  1. 具有对流核；
  2. 星风强，星风造成的物质流失严重；
  3. 比小质量恒星演化的快；
  4. C燃烧时不会出现剧烈的C闪耀现象；
  5. 中心氦燃烧阶段在水平巨星分支上不会打圈；
  6. 星风和内部对流使得大质量星在蓝、红超巨星间往复；
  7. 通过超新星爆发形成中子星或黑洞。
