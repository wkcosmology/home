Python内置库--configparser的使用

Posted on 2018-11-23 | In Note , Python
这篇文章主要介绍Python中配置文件读取工具--configparser库的使用。

configparse是Python中用来书写和解析配置文件的一个模块。通过使用配置文件的方式，我们可以在文件中保存我们的各种参数，这样子我们就不需要每次都改变源码或者是手动输入不同的参数。

config 文件的格式

  • 我们注意到，
    • [name]以及下面的一段称为一个section；
    • 每个section下面的键值对称为一个option
    • 支持注释，以;开头

读取config文件

['bitbucket.org', 'topsecret.server.com']
['user', 'serveraliveinterval', 'compression', 'compressionlevel', 'forwardx11']
